# Midjourney Prompt Catalog - Great White Icon Set

Use these prompts manually in Midjourney. They are tuned for strong silhouette readability after downscaling to 16x16 SVG icons.

## Global Template

```text
minimal vector icon, great white shark inspired, clean silhouette, bold shape language, high contrast, limited palette of steel blue teal icy white with optional complementary accent, centered composition, no text, no watermark, no border frame, no UI screenshot, no realistic photo texture, simple forms suitable for tiny file explorer icon, transparent or plain neutral background, flat or subtle gradient, 1:1
```

Recommended settings:
- Aspect ratio: `1:1`
- Export size target: `1024x1024` PNG source
- Keep detail low-to-medium so tracing remains crisp
