# Icon Reference Sources

Sample icons were pulled from these public locations during the Feb 2026 icon redesign session.
Do not commit the actual icon files — download on demand for reference only.

## Material Icon Theme (VS Code extension)
- **Source**: Local VS Code Insiders install
- **Path**: `%USERPROFILE%\.vscode-insiders\extensions\pkief.material-icon-theme-*\icons\`
- **License**: MIT — https://github.com/PKief/vscode-material-icon-theme
- **Used as reference for**: `learnings-folder.svg` style (folder body path + badge approach)
- **Key files reviewed**: `folder-docs.svg`, `folder-github.svg`

## Azure Public Service Icons
- **Source**: https://learn.microsoft.com/en-us/azure/architecture/icons/
- **Download**: https://aka.ms/AzureIconsPack (zip)
- **License**: Microsoft — see zip for terms; icons are free for use in Azure-related content
- **Conclusion**: 18×18 with thin strokes — too faint at 16×16; use for shape concepts only

## microsoft/vscode-icons
- **Source**: https://github.com/microsoft/vscode-icons
- **License**: CC BY 4.0 — https://creativecommons.org/licenses/by/4.0/
- **Download**: `gh repo clone microsoft/vscode-icons` or browse on GitHub
- **Key files reviewed**: `beaker.svg`, `book.svg`, `comment.svg`, `milestone.svg`,
  `star-full.svg`, `tasklist.svg`, `checklist.svg`, `bookmark.svg`,
  `note.svg`, `library.svg`, `history.svg`
